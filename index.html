Dogged is a .NET wrapper around libgit2. It was inspired by the LibGit2Sharp project.

API Reference

Dogged consists of two different projects that you can use to manage Git repositories.

Dogged

The Dogged project is a set of high-level APIs for working with Git repositories from .NET. These are the APIs that are recommended for all users.

Dogged.Native

The Dogged.Native project is a set of low-level PInvoke APIs that act as bindings for the libgit2 project. These APIs are used internally by Dogged, but are also provided for consumers who need to use low-level APIs.
